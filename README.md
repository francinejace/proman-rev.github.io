# proman-rev.github.io

This project provides an interactive reviewer for the **Project Management Certification Examination**. It converts a traditional reviewer into a quiz-style web application that allows users to test their knowledge and immediately see feedback on their answers.

## Overview

The reviewer contains **87 questions** based on the certification exam coverage. The application runs directly in the browser using **HTML, CSS, and JavaScript**, and is deployed using **GitHub Pages**.

Users can choose between two quiz modes:

* **Reviewer Mode** – Shows the correct answer and explanation immediately after selecting an option.
* **Exam Mode** – Simulates an exam environment by allowing users to answer questions without immediate feedback.

The interface displays one question per page and includes a progress indicator and score tracker.

## Features

* 87-question reviewer dataset
* One-question-per-page interface
* Reviewer Mode with instant explanations
* Exam Mode for self-assessment
* Progress bar to track quiz completion
* Score tracker
* Support for questions with multiple correct answers
* Clean and responsive layout

## How to Use

1. Open the deployed page.
2. Choose **Reviewer Mode** or **Exam Mode**.
3. Answer the questions one at a time.
4. Navigate using the **Next** and **Previous** buttons.
5. Track progress using the progress bar and score indicator.
